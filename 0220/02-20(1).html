03_부트스트랩(테이블관련)

부트스트랩 테이블 태그에는 약간의 셀 안쪽 여백 기본 적용.

table, caption, thead, tbody, tfoot, tr, th, td로 구성됨.

셀병합 - colspan, rowspan

class="table" : 테이블 기본 서식

class="table-striped" : 행마다 다은 색상을 적용

class="table-bordered" : 테이블 셀 테두리 만들기

class="table-hover" : 마우스 오버 시 배경색 변경

class="table-dark" : 테이블 제목줄(thead)에 배경색 어둡게 적용

class="table-success", class="table-danger", class="table-info", class="table-primary" : tr, td에 각각 분류 색상을 적용할 수 있음.

class="table-responsive" : 화면의 너비가 768px미만일 경우 가로로 스크롤됨.

페이지네이션 - pagination

nav, ul, li, a태그로 작성하고 nav태그에

정렬속성은 ul태그에 class='justify-content-center' 적용
