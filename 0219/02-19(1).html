문단 관련 서식

class - lead : 단락을 눈에 띄게 하기(글자 크기 : 20px)

class - small : 상위문자 크기의 85%=14px로 문자크기를 작게 설정함.

class - text-start : 문단정렬(오른쪽 정렬)

class - end : 문단정렬(왼쪽 정렬)

class - center : 문단정렬(가운데 정렬)

class - justify" : 문단정렬(양쪽배분 정렬)

class - nowrap" : 문단끝에서 글자가 다음줄로 넘어가지 않기

class - uppercase" : 문자를 대문자로 출력

class - lowercase" : 문자를 소문자로 출력

class - capitalize" : 영문자의 첫글자만 대문자로 출력
